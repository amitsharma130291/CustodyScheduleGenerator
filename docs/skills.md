# Skills

## Web Design Guidelines
- Follow docs/design.md.
- Linear-inspired UI.
- Dark and light mode required.
- Tool above the fold.
- Premium SaaS feel.
- Mobile-first.
- No generic calculator UI.

## Tailwind Agent Skills
- Use Tailwind utility classes.
- Prefer reusable components.
- Keep spacing consistent.
- Use CSS variables for themes.
- Avoid inline random colors.
- Use semantic tokens.

## Astro Guidelines
- Static pages by default.
- Use islands only when needed.
- Minimize JavaScript.
- Use semantic HTML.
- Optimize Core Web Vitals.

## SEO Guidelines
- One H1 per page.
- Tool above fold.
- FAQ section.
- Related internal links.
- Meta title and description.
- Schema where useful.
