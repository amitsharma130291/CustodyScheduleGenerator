# SEO Guidelines

## Goal

Build an SEO-first tool site that ranks for custody schedule and parenting time keywords.

## Page Rules

Every SEO page must include:

- One clear H1
- Keyword-focused title tag
- Meta description
- Canonical URL
- Tool above the fold
- Helpful intro paragraph
- FAQ section
- Related internal links
- Clean semantic HTML
- Fast mobile layout

## Page Structure

Use this structure for tool pages:

1. H1
2. Short intro
3. Generator/tool above the fold
4. Explanation section
5. Example schedule
6. FAQ
7. Related tools
8. Disclaimer
