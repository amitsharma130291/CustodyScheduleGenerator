# Product

## Product Name

Custody Schedule Generator

## Positioning

A free tool that helps divorced or separated parents create, preview, print, and download custody schedules.

## Core User Need

Users do not only want to read about custody schedules. They want to see their schedule visually and print it.

## Target Users

- Divorced parents
- Separated parents
- Family lawyers
- Mediators
- Co-parenting counselors

## MVP Features

- 2-2-3 schedule generator
- 2-2-5-5 schedule generator
- 5-2-2-5 schedule generator
- 3-4-4-3 schedule generator
- Alternating week schedule generator
- Calendar preview
- Print view
- PDF export

## Later Features

- Custody percentage calculator
- Holiday custody schedule generator
- Custody schedule by age
- Save schedule
- Share schedule
- Google Calendar export

## Monetization

Phase 1:
- AdSense

Phase 2:
- Better display ad network after traffic grows

Phase 3:
- Optional SaaS features

## Build Philosophy

- Free first
- No signup
- No backend initially
- No APIs
- Tool above the fold
- SEO pages around each schedule pattern
- Beautiful enough to feel like premium SaaS
